# Chrome Default Setup — Bookmarks Automation

This repository contains a **Chrome Bookmarks file** designed to streamline the setup of a fresh Google Chrome installation.
By importing this file, you can quickly open all the key settings pages to adjust Chrome’s privacy, permissions, and search defaults without manually navigating through menus.

---

## What This Repository Contains

* **`bookmarks.html`** – A Chrome-compatible bookmarks file.
  Importing it will add a folder with links to essential Chrome settings pages.

---

## Steps to Use

### 1. Import the Bookmarks File

1. Open Chrome.
2. Go to **Bookmarks → Bookmark Manager** (`Ctrl+Shift+O` or `Cmd+Option+B` on Mac).
3. Click the **⋮ (three-dot)** menu in the top-right corner.
4. Select **Import Bookmarks**.
5. Choose the `bookmarks.html` file from this repository.

---

### 2. Show the Bookmarks Bar

If the bookmarks bar isn’t visible:

* Press **Ctrl+Shift+B** (Windows/Linux) or **Cmd+Shift+B** (Mac),
  or go to **Settings → Appearance → Show Bookmarks Bar**.

---

### 3. Open All Tabs in the Setup Folder

1. Locate the imported **Setup / Chrome Defaults** folder in your bookmarks bar.
2. **Right-click** on the folder → select **“Open all (X)”** to open all settings pages at once.

---

### 4. Adjust Chrome’s Defaults

Go through each opened tab and update settings as needed. Recommended adjustments include:

* **Privacy & Security**

  * Disable ad personalization and telemetry features.
  * Turn off “Make searches and browsing better” (usage statistics).
* **Site Settings**

  * Set **Microphone** and **Camera** permissions to **Allow** for trusted sites.
  * Block unwanted pop-ups, notifications, and background sync.
* **Search Engine**

  * Remove **Yandex** and other unwanted search engines.
  * Set **Google** or your preferred provider as default.

---

## Optional Tips

* Sync your settings across devices using your Google account once configuration is complete.
* Export your customized bookmarks file for future setups (`Bookmark Manager → Export Bookmarks`).

---

## Purpose

This setup helps you quickly:

* Disable unnecessary data collection.
* Improve privacy and security.
* Configure site permissions.
* Remove unwanted defaults (like Yandex).
* Save time during new Chrome installations or profile resets.
